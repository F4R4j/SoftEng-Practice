# SoftEng-Practice

Program menghitung Volume air berdasarkan panjang lebar dan tinggi

program dalam bahasa C untuk menghitung volume air berdasarkan panjang, lebar, dan tinggi wadah (dalam satuan meter)

Langkah Langkah:
1 Program ini meminta pengguna untuk memasukkan panjang, lebar, dan tinggi air dalam meter.
2 Volume dihitung dengan rumus: volume = panjang × lebar × tinggi.
3 Hasil ditampilkan dalam satuan liter dengan dua angka di belakang koma.
